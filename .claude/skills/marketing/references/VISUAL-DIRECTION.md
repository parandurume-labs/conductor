## 1. 브랜드 컬러

### 메인 컬러

파란두루미는 이름(파란 + 두루미)에서 알 수 있듯이 **파란색 계열**이 브랜드의 핵심 색상입니다.

| 용도 | 색상명 | HEX | RGB | 설명 |
|---|---|---|---|---|
| **Primary** | 파란두루미 블루 | `#2B6CB0` | 43, 108, 176 | 메인 브랜드 컬러. 로고, 주요 CTA, 제목에 사용 |
| **Primary Dark** | 딥 블루 | `#1A4971` | 26, 73, 113 | 배경, 헤더, 강조 영역에 사용 |
| **Primary Light** | 스카이 블루 | `#63B3ED` | 99, 179, 237 | 보조 강조, 아이콘, 그래프 포인트에 사용 |

### 서브 컬러

| 용도 | 색상명 | HEX | RGB | 설명 |
|---|---|---|---|---|
| **Accent** | 화이트 | `#FFFFFF` | 255, 255, 255 | 텍스트, 배경, 여백 |
| **Accent Dark** | 차콜 그레이 | `#2D3748` | 45, 55, 72 | 본문 텍스트, 부제목 |
| **Accent Light** | 라이트 그레이 | `#EDF2F7` | 237, 242, 247 | 카드 배경, 구분선 영역 |
| **Point** | 테크 그린 | `#38B2AC` | 56, 178, 172 | 성공, 긍정적 수치, 포인트 강조 |
| **Alert** | 시그널 오렌지 | `#ED8936` | 237, 137, 54 | CTA 버튼, 긴급 알림, 주의 환기 |

### 컬러 사용 비율

```
Primary (파란 계열): 60%
Neutral (화이트 + 그레이): 30%
Accent (그린 + 오렌지): 10%
```

### 컬러 조합 규칙

1. **배경이 어두울 때**: 딥 블루 배경 + 화이트 텍스트 + 스카이 블루 강조
2. **배경이 밝을 때**: 화이트/라이트 그레이 배경 + 차콜 그레이 텍스트 + 파란두루미 블루 강조
3. **그라데이션**: 딥 블루 → 파란두루미 블루 → 스카이 블루 방향으로만 사용
4. **검정(#000000) 단독 사용 금지** — 차콜 그레이(#2D3748)로 대체

## 2. 폰트 가이드

### 권장 폰트

| 용도 | 한글 폰트 | 영문 폰트 | 대체 폰트 |
|---|---|---|---|
| **제목/헤드라인** | Pretendard Bold | Pretendard Bold | Noto Sans KR Bold |
| **소제목** | Pretendard SemiBold | Pretendard SemiBold | Noto Sans KR Medium |
| **본문** | Pretendard Regular | Pretendard Regular | Noto Sans KR Regular |
| **캡션/주석** | Pretendard Light | Pretendard Light | Noto Sans KR Light |
| **코드/기술 표기** | — | JetBrains Mono | Source Code Pro |

> **Pretendard**: 무료 오픈소스 폰트. 한글/영문 모두 지원하며, 가독성이 뛰어남.
> 다운로드: https://cactus.tistory.com/306

### 폰트 크기 가이드

**영상 콘텐츠 (썸네일, 숏폼 자막):**

| 요소 | 크기 | 비고 |
|---|---|---|
| 메인 타이틀 | 48~72pt | 화면의 40% 이내 차지 |
| 서브 타이틀 | 32~40pt | |
| 자막 | 28~36pt | 하단 중앙 배치, 배경 반투명 처리 |
| 워터마크/로고 | 16~20pt | 우측 상단, 불투명도 60~70% |

**카드뉴스 / SNS 이미지:**

| 요소 | 크기 | 비고 |
|---|---|---|
| 헤드라인 | 36~48pt | Bold, 최대 2줄 |
| 본문 텍스트 | 20~28pt | Regular, 최대 6줄 |
| 캡션/출처 | 14~18pt | Light |
| 페이지 번호 | 14pt | 하단 중앙 또는 우측 |

### 폰트 사용 규칙

1. **한 디자인에 2개 이상 폰트 종류 혼용 금지** — 같은 폰트 패밀리 내에서 굵기로 구분
2. **자간(Letter Spacing)**: 제목 -2~0%, 본문 0~2%
3. **행간(Line Height)**: 제목 120~140%, 본문 160~180%
4. **영문 대문자 단독 사용**: 브랜드명(DURU BIRD 등)과 약어(XR, CES)에만 허용

## 3. 카드뉴스 레이아웃 가이드

### 기본 그리드

```
┌──────────────────────────┐
│        Safe Zone         │
│  ┌──────────────────┐    │
│  │                  │    │
│  │    Content       │    │
│  │    Area          │    │
│  │                  │    │
│  └──────────────────┘    │
│                          │
└──────────────────────────┘

전체 캔버스: 1080 x 1350px (4:5)
안전 영역(Safe Zone) 여백: 상하좌우 각 80px
콘텐츠 영역: 920 x 1190px
```

### 여백 규칙

| 여백 유형 | 크기 | 설명 |
|---|---|---|
| **외부 여백 (Margin)** | 80px | 캔버스 테두리 ~ 콘텐츠 영역 |
| **내부 여백 (Padding)** | 40px | 콘텐츠 블록 간 간격 |
| **텍스트 간 여백** | 20~30px | 제목-본문, 문단-문단 사이 |
| **이미지-텍스트 간 여백** | 30px | 이미지와 설명 텍스트 사이 |

### 카드뉴스 레이아웃 유형

**유형 A: 타이틀 카드 (표지)**
```
┌────────────────────┐
│                    │
│   [브랜드 로고]     │
│                    │
│   큰 제목 텍스트    │
│   (최대 2줄)       │
│                    │
│   부제목/요약       │
│                    │
│   파란두루미        │
└────────────────────┘
```
- 배경: 딥 블루 또는 브랜드 이미지
- 텍스트: 화이트
- 로고: 상단 중앙 또는 좌측 상단

**유형 B: 텍스트 중심**
```
┌────────────────────┐
│   소제목           │
│   ─────────        │
│                    │
│   본문 텍스트       │
│   (최대 6줄)       │
│                    │
│   핵심 포인트       │
│   강조 박스         │
│                    │
│          3/8       │
└────────────────────┘
```
- 배경: 화이트 또는 라이트 그레이
- 소제목: 파란두루미 블루
- 본문: 차콜 그레이

**유형 C: 이미지 + 텍스트**
```
┌────────────────────┐
│                    │
│   [이미지/스크린샷]  │
│   (화면 50~60%)    │
│                    │
│   ─────────        │
│   설명 텍스트       │
│   (최대 3줄)       │
│                    │
│          5/8       │
└────────────────────┘
```
- 이미지: 둥근 모서리(Radius 12px) 적용
- 이미지 그림자: `0px 4px 16px rgba(0,0,0,0.1)`

**유형 D: 숫자/데이터 강조**
```
┌────────────────────┐
│                    │
│   소제목           │
│                    │
│      40%           │
│   유지보수 시간     │
│      절감          │
│                    │
│   설명 텍스트       │
│                    │
│          6/8       │
└────────────────────┘
```
- 숫자: Pretendard Bold 72pt, 파란두루미 블루 또는 테크 그린
- 단위/설명: Regular 24pt

### 카드뉴스 시리즈 구성 (권장)

| 순서 | 유형 | 내용 |
|---|---|---|
| 1장 | 유형 A (표지) | 주제 + 브랜드 |
| 2~3장 | 유형 B or C | 문제 제기 또는 배경 설명 |
| 4~6장 | 유형 C or D | 핵심 내용, 데이터, 사례 |
| 7장 | 유형 B | 정리/요약 |
| 8장 | 유형 A 변형 | CTA + 연락처/링크 |

## 4. 썸네일 디자인 원칙

### 유튜브 숏츠 / 릴스 / 틱톡 커버

**규격**: 1080 x 1920px (9:16)

**구성 원칙:**

1. **텍스트 배치**: 화면 상단 1/3에 메인 텍스트 (하단은 플랫폼 UI에 가려짐)
2. **텍스트 최대 10자** — 한눈에 읽히는 짧은 문구
3. **배경**: 실제 장면 캡처 + 반투명 오버레이(딥 블루, 불투명도 40~60%)
4. **텍스트 가독성**: 텍스트 뒤에 그림자 또는 반투명 배경 박스 처리
5. **로고**: 우측 상단, 작게

### 네이버 블로그 대표 이미지

**규격**: 1200 x 630px (1.91:1)

**구성 원칙:**

1. 좌측 60%에 핵심 이미지(제품 스크린샷, 현장 사진)
2. 우측 40%에 제목 텍스트
3. 하단에 파란두루미 로고 + 브랜드 컬러 띠
4. 제목은 최대 15자

### LinkedIn / SNS 이미지

**규격**: 1200 x 627px (1.91:1) 또는 1080 x 1080px (1:1)

**구성 원칙:**

1. 깔끔하고 프로페셔널한 느낌 유지
2. 텍스트 최소화 — 핵심 메시지 1줄 + 데이터 1개
3. 브랜드 컬러 그라데이션 배경 또는 화이트 배경
4. 인물 사진 사용 시 자연스러운 업무 장면 (연출된 포즈 사진 지양)

## 5. 비주얼 Do / Don't

### DO (이렇게 하세요)

| 번호 | 원칙 | 설명 |
|---|---|---|
| 1 | **일관된 컬러 사용** | 모든 콘텐츠에서 브랜드 컬러 팔레트를 지키세요. 콘텐츠를 모아 보았을 때 "파란두루미"임을 한눈에 알 수 있어야 합니다. |
| 2 | **충분한 여백 확보** | 여백은 고급스러움의 핵심입니다. 요소 간 간격을 넉넉히 두세요. |
| 3 | **고해상도 이미지 사용** | 흐릿하거나 픽셀이 깨진 이미지는 절대 사용하지 마세요. 최소 72dpi (웹), 인쇄물은 300dpi. |
| 4 | **실제 제품 화면 활용** | DURU BIRD, DURU TWIN 등의 실제 스크린샷과 데모 영상을 적극 활용하세요. 스톡 이미지보다 진정성 있습니다. |
| 5 | **텍스트 계층 구조 유지** | 제목 > 소제목 > 본문 > 캡션 순서의 시각적 위계를 항상 유지하세요. |
| 6 | **둥근 모서리 통일** | UI 스크린샷, 카드, 이미지 프레임에 Radius 8~16px 적용. 날카로운 직각보다 부드러운 인상. |
| 7 | **아이콘 스타일 통일** | 라인 아이콘(선형) 또는 플랫 아이콘 중 하나로 통일. 혼용 금지. |
| 8 | **다크/라이트 모드 대비 확인** | 텍스트-배경 간 명암 대비를 WCAG AA 기준(4.5:1) 이상으로 유지하세요. |

### DON'T (이렇게 하지 마세요)

| 번호 | 금지 사항 | 이유 |
|---|---|---|
| 1 | **브랜드 컬러 이외의 원색 사용** | 빨강, 노랑 등 지정 외 원색은 브랜드 이미지를 해칩니다. |
| 2 | **텍스트 과다 배치** | 카드뉴스 한 장에 텍스트가 화면의 50%를 넘지 않도록. 핵심만 담으세요. |
| 3 | **저해상도 이미지, 워터마크 있는 스톡 이미지** | 무료 스톡 사이트의 워터마크가 남은 이미지는 브랜드 신뢰도를 떨어뜨립니다. |
| 4 | **그림자/효과 과다 사용** | 드롭 섀도우, 글로우, 베벨 등을 겹겹이 쌓지 마세요. 그림자는 최소한으로. |
| 5 | **한 이미지에 3가지 이상 폰트 사용** | 산만해 보입니다. 하나의 폰트 패밀리, 굵기 변화로만 구분하세요. |
| 6 | **로고 비율 변형** | 로고를 늘리거나 찌그러뜨리지 마세요. 항상 원본 비율 유지. |
| 7 | **배경과 유사한 색상의 텍스트** | 밝은 배경에 밝은 텍스트, 어두운 배경에 어두운 텍스트는 가독성을 해칩니다. |
| 8 | **다른 회사 로고를 무단 사용** | 파트너/고객사 로고 사용 시 반드시 사전 승인을 받으세요. |
| 9 | **과도한 애니메이션/전환 효과** | 영상에서 화려한 전환 효과보다 콘텐츠 자체에 집중하세요. 전환은 깔끔한 컷 또는 페이드 위주로. |
| 10 | **비공식 로고/색상 조합** | 임의로 만든 로고 변형이나 승인되지 않은 색상 조합을 사용하지 마세요. |
