## 1. 유튜브 숏츠 (YouTube Shorts)

### 영상 규격

| 항목 | 규격 |
|---|---|
| **해상도** | 1080 x 1920px (풀HD 세로) |
| **화면 비율** | 9:16 (세로 필수) |
| **영상 길이** | 최소 15초 ~ 최대 60초 (30~45초 권장) |
| **파일 형식** | MP4, MOV |
| **최대 파일 크기** | 256MB |
| **프레임 레이트** | 30fps 이상 (60fps 권장) |

### 썸네일 가이드

- 숏츠는 자동 썸네일이 기본이지만, 채널 피드에서 표시되므로 **첫 장면을 썸네일처럼 설계**할 것
- 첫 프레임에 **큰 텍스트 + 강렬한 비주얼** 배치
- 텍스트는 화면의 상단 1/3에 배치 (하단은 UI 요소에 가려짐)
- 파란두루미 로고 워터마크: 우측 상단, 불투명도 70%, 크기 화면의 5% 이내

### 최적 게시 시간 (한국 기준)

| 시간대 | 적합도 | 설명 |
|---|---|---|
| **오전 7:00~9:00** | 높음 | 출근길 모바일 시청 |
| **오후 12:00~13:00** | 높음 | 점심시간 시청 |
| **오후 18:00~21:00** | 매우 높음 | 퇴근 후 가장 활발한 시간대 |
| **주말 오전 10:00~12:00** | 보통 | 여유로운 탐색 시간 |

### 알고리즘 팁

1. **첫 2초가 승부** - 스크롤을 멈추게 하는 후킹(hooking)이 핵심. 질문, 놀라운 장면, 강렬한 텍스트로 시작
2. **끝까지 시청률(완주율)이 가장 중요한 지표** - 60초보다 30초 영상의 완주율이 높음. 짧게 임팩트 있게
3. **루프(반복 시청) 구조** 활용 - 영상 끝이 영상 시작과 자연스럽게 이어지도록 편집
4. **자막 필수** - 무음 시청 비율이 70% 이상. 크고 읽기 쉬운 자막 삽입
5. **게시 후 1시간 이내 댓글 대응** - 초기 인게이지먼트가 노출 확대에 결정적
6. **일관된 게시 스케줄** 유지 - 알고리즘이 정기적 업로더를 우대
7. **트렌딩 오디오** 적극 활용 - 인기 있는 BGM이나 효과음 사용 시 도달 증가

1. **릴스 게시 후 스토리로 공유** - "새 릴스 올렸어요!" 스티커와 함께 스토리에 재공유
2. **스토리 하이라이트 카테고리** 정리:

3. **스토리 인터랙션 스티커 활용** - 설문, 퀴즈, 질문 스티커로 참여 유도
4. **스토리 → 릴스 유도 흐름** 만들기: 스토리에서 티저 → 릴스에서 본편
5. **일주일에 최소 5일** 스토리 업로드 — 스토리 꾸준히 올리는 계정이 알고리즘 우대

### 트렌드 활용법

1. **트렌딩 사운드 모니터링** - 매일 "검색 > 트렌드" 탭에서 떠오르는 오디오 확인
2. **트렌드 템플릿 활용** - 유행하는 편집 스타일(전환 효과, 텍스트 노출 방식)을 XR 콘텐츠에 접목
3. **트렌드 반응 속도가 핵심** - 트렌드 발견 후 24~48시간 이내 제작·게시
4. **파란두루미만의 시리즈 해시태그** 만들기: `#파란두루미XR` `#이것도XR로된다고`
5. **밈(meme) 활용** - XR 기술의 신기한 순간을 밈 형식으로 가공

### 최적 게시 시간 (한국 기준)

|---|---|
| **오전 7:00~9:00** | 보통 |
| **오후 12:00~13:00** | 높음 |
| **오후 19:00~22:00** | 매우 높음 |
| **금요일~일요일** | 전체적으로 높음 |

### 듀엣/스티치 활용

1. **듀엣(Duet)** — 화면을 반으로 나눠 반응 영상 제작
   - 활용 예: 다른 XR/기술 계정의 영상에 파란두루미 시점으로 리액션
   - 활용 예: 자사 이전 영상에 "그 후 어떻게 됐냐면..." 식의 후속편
2. **스티치(Stitch)** — 다른 영상의 앞부분(최대 5초)을 가져와 이어 촬영
   - 활용 예: "XR이 뭔지 모른다고?" → 파란두루미 데모 영상으로 전환
   - 활용 예: 기술 관련 질문 영상에 전문가 답변 형식으로 연결
3. **듀엣/스티치 허용 설정**: 자사 영상에 대해 반드시 허용으로 설정 — 바이럴 기회 확대

## 4. 네이버 블로그

### SEO 키워드 전략

**핵심 키워드 (매 게시물에 1~2개 포함):**
```
디지털트윈, XR솔루션, 확장현실, 디지털트윈 대시보드, 핸드트래킹
```

**롱테일 키워드 (검색 의도에 맞춰 선택):**
```
디지털트윈 활용 사례, XR 도입 효과, 제조업 디지털 전환,
스마트팩토리 XR, 건설현장 디지털트윈, XR 원격 협업,
디지털트윈 대시보드 구축, 핸드트래킹 기술 원리
```

**키워드 배치 원칙:**
1. **제목**: 핵심 키워드를 앞쪽에 자연스럽게 포함 (예: "디지털트윈 대시보드 활용 사례 — DURU TWIN으로 현장을 한눈에")
2. **본문 첫 문단**: 핵심 키워드 1회 이상 포함
3. **소제목(H2/H3)**: 키워드 포함된 소제목 2~3개
4. **본문 전체**: 키워드 밀도 2~3% (과도한 반복 금지)
5. **태그**: 핵심 + 롱테일 조합 10~15개

### 최적 글 길이

| 글 유형 | 권장 길이 | 이미지 수 |
|---|---|---|
| **제품 소개** | 1,500~2,500자 | 5~10장 |
| **고객 사례** | 2,000~3,000자 | 5~8장 + 인포그래픽 1장 |
| **기술 해설** | 2,000~3,500자 | 3~7장 + 도표 |
| **전시/이벤트 후기** | 1,500~2,000자 | 8~15장 |
| **뉴스/공지** | 800~1,200자 | 2~5장 |

### 이미지 배치 가이드

1. **글 시작에 대표 이미지(커버)** 반드시 배치 — 검색 결과 썸네일로 표시
2. **2~3문단마다 이미지 1장** 삽입 — 텍스트만 나열되면 이탈률 급증
3. **이미지 크기**: 가로 기준 최소 750px 이상 (모바일에서도 선명하게)
4. **이미지 ALT 텍스트**: 키워드 포함한 설명 입력 (예: "DURU TWIN 디지털트윈 대시보드 화면")
5. **인포그래픽/도표**: 복잡한 내용은 시각화하여 전달
6. **영상 삽입**: 유튜브 영상 임베드 시 블로그 체류 시간 증가 → 검색 순위 상승

### 검색 노출 팁

1. **발행 후 24시간 이내** 네이버 서치어드바이저에서 수동 색인 요청
2. **주 1~2회 꾸준한 발행** — 불규칙 발행보다 일정한 주기가 중요
3. **이웃 소통** — 관련 블로그와 상호 방문·댓글로 블로그 지수(C-Rank) 향상
4. **본문에 관련 내부 링크** 삽입 — 자사 이전 글 연결로 체류 시간 증가
5. **모바일 최적화** — 문단 3~4줄 이내, 이미지 가로폭 100%, 긴 문장 피하기
6. **스마트블록 노출 노리기** — "OO이란?", "OO 방법" 등 질문형 소제목 활용

| **네이버 블로그 (장문 글)** | 핵심 요약 → LinkedIn 포스트 / 카드뉴스화 → Instagram 캐러셀 |

### 크로스 포스팅 시 주의사항

1. **각 플랫폼에 맞게 편집** — 동일 영상이라도 자막 크기, 캡션 톤을 채널별로 조정
2. **TikTok 워터마크가 있는 영상을 Instagram에 올리지 않기** — 알고리즘 불이익
3. **게시 시간 차이 두기** — 같은 콘텐츠를 동시에 올리지 말고 1~2시간 간격
4. **채널별 CTA 다르게** — 유튜브는 "구독", Instagram은 "팔로우", 블로그는 "문의하기"
