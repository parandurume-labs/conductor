## 1. 타겟 페르소나

### 페르소나 A: B2B 의사결정자 ("김 팀장")

| 항목 | 내용 |
|---|---|
| **직함** | 제조·건설·공공기관 중간관리자 ~ 임원 (팀장, 부서장, CTO) |
| **나이대** | 35~55세 |
| **핵심 니즈** | 현장 효율화, 비용 절감, 디지털 전환 성과 입증 |
| **정보 탐색 경로** | LinkedIn, 네이버 블로그 검색, 산업 전시회(CES 등), 동료 추천 |
| **구매 결정 요소** | ROI 수치, 레퍼런스 사례, 기술 신뢰도, 유지보수 지원 체계 |
| **페인포인트** | "XR이 좋다는데 우리 현장에 실제로 적용 가능한가?", "예산 대비 효과를 경영진에게 어떻게 보고하지?" |
| **우리가 제공할 가치** | 현장 적용 중심 XR 솔루션 + 도입 전후 비교 데이터 + 신뢰 기반 파트너십 |
| **콘텐츠 선호** | 고객 사례 보고서, ROI 인포그래픽, 기술 백서, 웨비나 |

### 페르소나 B: 기술 관심 일반인 ("이 대리")

| 항목 | 내용 |
|---|---|
| **직함** | IT·디자인·콘텐츠 분야 직장인 또는 대학(원)생 |
| **나이대** | 22~38세 |
| **핵심 니즈** | 최신 기술 트렌드 파악, "이걸로 뭘 할 수 있지?" 호기심 충족 |
| **정보 탐색 경로** | YouTube Shorts, Instagram Reels, TikTok, 브런치/미디엄 |
| **관심 트리거** | 신기한 데모 영상, 몰입감 있는 체험 후기, 쉬운 기술 해설 |
| **페인포인트** | "XR 기술이 어렵게 느껴진다", "일상이나 내 업무에 어떤 영향을 줄지 모르겠다" |
| **우리가 제공할 가치** | 누구나 이해할 수 있는 XR 콘텐츠 + 체험 기회 안내 |
| **콘텐츠 선호** | 30~60초 숏폼 영상, 비하인드 스토리, 트렌드 해설, 체험 이벤트 |

### 페르소나 C: 잠재 채용 후보 ("박 개발자")

| 항목 | 내용 |
|---|---|
| **직함** | XR·3D·Unity·프론트엔드 개발자, UX 디자이너, 기획자 |
| **나이대** | 24~40세 |
| **핵심 니즈** | 성장 가능성 있는 팀, 의미 있는 프로젝트, 좋은 동료 |
| **정보 탐색 경로** | LinkedIn, 브런치/미디엄, GitHub, 채용 플랫폼 |
| **관심 트리거** | 팀 문화 엿보기, 기술 스택 공개, 대표·팀원 인터뷰, 사회적 가치 |
| **페인포인트** | "작은 회사인데 성장할 수 있을까?", "XR 업계 전망이 궁금하다" |
| **우리가 제공할 가치** | 임팩트 중심 제품 사고 + 예비사회적기업으로서의 비전 + 실제 팀 분위기 |
| **콘텐츠 선호** | 팀 브이로그, 기술 블로그, 프로젝트 회고, 채용 공고 + 스토리 |

## 2. 채널별 전략

### 2-1. YouTube Shorts

| 항목 | 내용 |
|---|---|
| **목적** | 브랜드 인지도 확산, "이 회사 뭐하는 곳이지?" 호기심 유발 |
| **주 타겟** | 페르소나 B (기술 관심 일반인) |
| **콘텐츠 유형** | XR 데모 하이라이트 (30~60초), DURU BIRD 핸드트래킹 체험 영상, "이것도 XR로 된다고?" 시리즈, 전시회(CES 등) 현장 스케치 |
| **게시 빈도** | 주 2~3회 |
| **KPI** | 조회수 (월 10,000+), 구독자 증가율 (월 5%+), 평균 시청 시간 (15초+) |
| **팁** | 첫 2초가 승부. 자막 필수. 세로 영상(9:16). 끝에 "더 보기" 유도 |

### 2-2. Instagram Reels

| 항목 | 내용 |
|---|---|
| **목적** | 브랜드 감성 전달, 팀 문화·비하인드 공유 |
| **주 타겟** | 페르소나 B + C |
| **콘텐츠 유형** | 제품 시연 릴스, 팀 일상/오피스 투어, 프로젝트 비하인드, 이벤트·전시 현장 |
| **게시 빈도** | 주 2~3회 (릴스 + 피드/스토리 병행) |
| **KPI** | 릴스 도달 수 (월 20,000+), 프로필 방문 수, 팔로워 증가율 (월 3%+) |
| **팁** | 트렌드 오디오 활용. 캐러셀(여러 장 슬라이드)도 적극 활용. 스토리로 일상 소통 |

### 2-3. TikTok

| 항목 | 내용 |
|---|---|
| **목적** | MZ세대 도달, 바이럴 콘텐츠 실험 |
| **주 타겟** | 페르소나 B |
| **콘텐츠 유형** | "XR 기술 놀라운 순간" 모음, 트렌드 챌린지에 XR 접목, 직원 참여형 콘텐츠, 유머·밈 활용 기술 소개 |
| **게시 빈도** | 주 2~4회 |
| **KPI** | 영상별 조회수 (5,000+), 좋아요·공유 수, 팔로워 1,000명 달성 시점 |
| **팁** | TikTok은 실험의 장. 완성도보다 속도와 재미 우선. 실패해도 OK |

### 2-4. 네이버 블로그

| 항목 | 내용 |
|---|---|
| **목적** | 검색 유입(SEO), B2B 신뢰 구축, 상세 정보 제공 |
| **주 타겟** | 페르소나 A (B2B 의사결정자) |
| **콘텐츠 유형** | 제품 상세 소개 포스트, 고객 사례 인터뷰, XR/디지털트윈 용어 해설, 전시회 참가 후기, MOU·수상 소식 |
| **게시 빈도** | 주 1~2회 |
| **KPI** | 네이버 검색 노출 키워드 수, 블로그 방문자 수 (일 100+), 문의 전환 수 |
| **팁** | 제목에 핵심 키워드 포함 (예: "디지털트윈 대시보드 활용 사례"). 이미지·도표 풍부하게. 본문 1,500자 이상 권장 |

### 2-5. LinkedIn

| 항목 | 내용 |
|---|---|
| **목적** | B2B 네트워킹, 글로벌 브랜딩, 채용 브랜딩 |
| **주 타겟** | 페르소나 A + C |
| **콘텐츠 유형** | 회사 마일스톤 발표 (CES, MOU 등), 대표·팀원 인사이트 포스트, 산업 동향 코멘터리, 채용 공고 + 팀 소개, 영문 콘텐츠 (글로벌 타겟 시) |
| **게시 빈도** | 주 2~3회 |
| **KPI** | 게시물 노출수 (월 5,000+), 연결(Connection) 증가, 인바운드 문의 수 |
| **팁** | 개인 계정(대표, 팀원)에서도 공유하면 도달 2~3배 증가. 텍스트 중심 포스트가 LinkedIn에서는 성과 좋음 |

### 2-6. 브런치 / 미디엄 (Medium)

| 항목 | 내용 |
|---|---|
| **목적** | 깊이 있는 스토리텔링, 사고 리더십(Thought Leadership), 채용 브랜딩 |
| **주 타겟** | 페르소나 B + C |
| **콘텐츠 유형** | 프로젝트 회고록, XR 산업 전망 에세이, "왜 우리는 이 제품을 만드는가" 시리즈, 팀원 인터뷰·성장기, 기술적 의사결정 과정 공유 |
| **게시 빈도** | 격주 1회 (월 2회) |
| **KPI** | 글 조회수 (500+/편), 공유·추천 수, 채용 지원 시 "브런치 보고 왔어요" 언급 |
| **팁** | 브런치는 감성 + 스토리, 미디엄은 기술 + 인사이트. 용도에 맞게 분리 운영 가능 |

## 4. 톤앤매너 가이드

### 기본 원칙

> **흥미 유발 + 신뢰감. 격식 없이 친근하되 전문성이 느껴지는 톤.**

우리는 "어려운 기술을 쉽고 매력적으로 전달하는 회사"입니다. 모든 콘텐츠에서 이 느낌을 유지하세요.

### DO (이렇게 하세요)

| 번호 | 가이드 | 예시 |
|---|---|---|
| 1 | **쉬운 말로 설명하세요.** 전문 용어를 쓰더라도 바로 옆에 쉬운 풀이를 넣으세요. | "디지털 트윈(현실을 그대로 복제한 가상 모델)으로..." |
| 2 | **구체적 숫자와 사례를 제시하세요.** 막연한 표현 대신 근거를 보여주세요. | "유지보수 시간이 40% 단축되었습니다" (O) vs "효율이 크게 올랐습니다" (X) |
| 3 | **"~해요" 체를 기본으로 사용하세요.** 블로그·SNS에서는 해요체, LinkedIn·보도자료에서는 합니다체. | "이번 CES에서 새로운 데모를 선보였어요." |
| 4 | **궁금증을 유발하는 질문으로 시작하세요.** | "손만 움직여서 건물을 설계할 수 있다면?" |
| 5 | **우리 팀의 진심과 과정을 보여주세요.** 완벽한 결과만이 아니라 고민과 시행착오도 공유하세요. | "처음엔 이 기능이 잘 안 돼서 3주를 매달렸어요." |
| 6 | **비주얼을 최우선으로 활용하세요.** XR 회사답게 영상·이미지·GIF가 텍스트보다 먼저 와야 합니다. | 글 시작에 데모 GIF 삽입 |
| 7 | **행동 유도(CTA)를 명확히 넣으세요.** | "더 자세한 내용은 프로필 링크에서 확인하세요!" |
| 8 | **사회적 가치를 자연스럽게 녹이세요.** 과하지 않게, 우리가 왜 이 일을 하는지 보여주세요. | "기술이 더 포용적인 세상을 만든다고 믿어요." |

### DON'T (이렇게 하지 마세요)

| 번호 | 가이드 | 예시 |
|---|---|---|
| 1 | **과장하지 마세요.** "세계 최고", "혁신적인", "유일무이한" 같은 근거 없는 수식어를 피하세요. | "업계 유일의 솔루션" (X) |
| 2 | **경쟁사를 직접 비교하거나 깎아내리지 마세요.** | "A사와 달리 우리는..." (X) |
| 3 | **지나치게 딱딱한 보고서 톤을 쓰지 마세요.** SNS에서는 특히 주의. | "본 솔루션은 제반 업무 프로세스의 혁신적 개선을 도모합니다" (X) |
| 4 | **확인되지 않은 정보를 게시하지 마세요.** 수치·사례는 반드시 팩트체크 후 게시. | |
| 5 | **너무 긴 텍스트만 올리지 마세요.** 숏폼 채널에서 장문은 역효과. | TikTok에 300자 설명문 (X) |
| 6 | **이모지를 과도하게 사용하지 마세요.** 포인트로만 활용. 문장마다 이모지 금지. | "우리 팀 최고!! 대박!! 짱!!" (X) |
| 7 | **고객·파트너 정보를 무단 공개하지 마세요.** 사례 공유 시 반드시 사전 동의 확인. | |
| 8 | **정치·종교·논쟁적 이슈에 대한 의견을 표명하지 마세요.** | |
