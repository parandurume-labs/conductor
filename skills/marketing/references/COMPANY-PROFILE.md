## 2. 제품/서비스 상세

### 2.1 XR SOLUTION — 산업 현장을 위한 XR 기반 교육·협업 솔루션

- **개요:** 산업 현장의 교육과 협업을 XR 기술로 혁신하는 통합 솔루션
- **주요 기능:**
  - 산업 현장 맞춤형 XR 교육 콘텐츠 제공
  - 원격 협업 및 현장 시뮬레이션
  - 팀 기반 협업 기능
- **타겟 고객:** 제조업, 산업 현장, 안전 교육이 필요한 기업
- **핵심 가치:** 위험한 현장 환경을 가상으로 재현하여 안전하고 효율적인 교육 실현

### 2.2 DURU BIRD — 직관적 XR 경험 플랫폼

- **개요:** 핸드 트래킹과 멀티모달 인터랙션으로 구현하는 직관적 XR 경험
- **핵심 기술:**
  - 핸드 트래킹 (Hand Tracking): 컨트롤러 없이 손동작만으로 자연스러운 상호작용
  - 멀티모달 인터랙션 (Multimodal Interaction): 시각, 청각, 촉각 등 복합 감각 활용
- **타겟 고객:** 자연스러운 XR 상호작용을 원하는 기업 및 기관
- **차별점:** 컨트롤러 의존도를 낮춰 XR 진입 장벽을 획기적으로 감소

### 2.3 DURU TWIN — 디지털 트윈 대시보드

- **개요:** 실시간 현장 데이터를 시각화하는 디지털 트윈 대시보드
- **주요 기능:**
  - 실시간 데이터 수집 및 시각화
  - 현장 모니터링 대시보드
  - 운영 데이터 분석 및 인사이트 제공
- **타겟 고객:** 시설 관리, 운영팀, 스마트 팩토리 관리자
- **핵심 가치:** 물리적 현장을 디지털로 복제하여 원격에서도 실시간 상황 파악 가능

### 2.4 3D INTERACTIVE — 웹 기반 3D 인터랙티브 콘텐츠

- **개요:** 별도 설치 없이 웹 브라우저에서 실행되는 3D 인터랙티브 콘텐츠
- **주요 기능:**
  - 웹 기반 3D 시각화
  - 인터랙티브 프레젠테이션
  - 크로스 플랫폼 접근성
- **타겟 고객:** 3D 콘텐츠가 필요한 기업, 교육 기관, 마케팅 팀
- **핵심 가치:** 접근성 — 누구나 브라우저만 있으면 3D 콘텐츠를 경험

### 2.5 DURU CASE MANAGER — 공동소송 관리 플랫폼

- **개요:** 공동소송 원고 데이터를 안전하게 수집·관리하는 운영 플랫폼
- **주요 기능:**
  - 원고 데이터 보안 수집
  - 소송 데이터 체계적 관리
  - 운영 자동화
- **타겟 고객:** 법무법인, 공동소송 운영 조직
- **핵심 가치:** 민감한 소송 데이터를 안전하고 효율적으로 관리

## 6. 차별화 포인트

### 기술적 차별화
- **핸드 트래킹 기반 XR**: 컨트롤러 없는 자연스러운 인터랙션으로 XR 진입 장벽 해소
- **패시브 스테레오 XR 시스템**: 독자적 기술 접근으로 비용 효율적 XR 환경 구축
- **멀티모달 인터랙션**: 단일 감각이 아닌 복합 감각 기반 UX 설계
- **웹 기반 3D**: 별도 앱 설치 없이 브라우저에서 즉시 접근 가능

### 비즈니스 차별화
- **현장 밀착형 개발**: 실험실이 아닌 산업 현장에서 출발하는 제품 개발 방식
- **풀스택 제품군**: XR, 디지털 트윈, 3D 콘텐츠, 데이터 관리까지 원스톱 제공
- **의료-산업 크로스 도메인**: 길병원 MOU 등 의료와 산업 양쪽에서 검증
- **글로벌 무대 검증**: CES 2025 참가로 글로벌 시장 진출 기반 확보

### 사회적 가치 차별화
- **예비사회적기업**: 기술 기업이면서 사회적 가치를 추구하는 독특한 포지셔닝
- **ESG 실천**: 포용적 기술, 사회연대경제를 통한 지속가능한 커뮤니티 기여
- **신뢰 기반 파트너십**: 광명시, 의료기관 등 공공-민간 협력 모델 구축
