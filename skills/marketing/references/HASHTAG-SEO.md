## 1. 브랜드 해시태그

파란두루미의 모든 게시물에 **반드시 1~2개 이상** 포함해야 하는 고유 해시태그입니다.

### 필수 브랜드 해시태그 (한글)

```
#파란두루미
#두루버드
#두루트윈
#두루케이스매니저
```

### 필수 브랜드 해시태그 (영문)

```
#Parandurume
#DURUBIRD
#DURUTWIN
#DURUCASEMANAGER
```

### 브랜드 슬로건 해시태그

```
#현실을넘어서
#기술로연결하는세상
#XR로만드는내일
#파란두루미XR
```

### 사용 규칙

| 규칙 | 설명 |
|---|---|
| **항상 첫 번째에 배치** | `#파란두루미`는 모든 해시태그 세트의 가장 앞에 위치 |
| **한글+영문 병행** | 국내 채널은 한글 우선, LinkedIn/글로벌은 영문 우선 |
| **표기 통일** | `#파란두루미` (O), `#파란 두루미` (X), `#파란_두루미` (X) |
| **제품명은 영문 유지** | DURU BIRD, DURU TWIN 등 제품명은 영문 대문자 표기 |

## 3. 채널별 해시태그 전략

각 소셜 미디어 채널마다 해시태그의 최적 개수, 배치 방법, 알고리즘 특성이 다릅니다.

### 3-1. YouTube (Shorts 포함)

| 항목 | 전략 |
|---|---|
| **최적 개수** | 5~10개 |
| **배치 위치** | 영상 설명(Description)란에 기재 |
| **제목 해시태그** | 제목 앞 또는 뒤에 핵심 해시태그 1~2개 추가 가능 (예: `#XR 손으로 건물을 만든다고?`) |
| **태그(Tags) 기능** | YouTube Studio 업로드 시 '태그' 필드에 관련 키워드 10~15개 별도 입력 |
| **Shorts 특이사항** | `#Shorts` 태그 필수 포함. 설명란에 3~5개 해시태그로 간결하게 |
| **알고리즘 팁** | YouTube는 해시태그보다 **제목, 설명 본문의 자연어 키워드**가 더 중요. 해시태그는 보조 수단 |

**배치 예시 (Shorts):**
```
손만 움직여서 건물을 설계한다고? #Shorts

DURU BIRD의 핸드트래킹 기술을 직접 체험해보세요!

#파란두루미 #DURUBIRD #핸드트래킹 #XR솔루션 #확장현실
```

### 3-2. Instagram (Reels/피드/스토리)

| 항목 | 전략 |
|---|---|
| **최적 개수** | 15~20개 (최대 30개 허용, but 20개 이내 권장) |
| **배치 위치** | 캡션 본문 아래 줄바꿈 후 배치 또는 첫 번째 댓글에 배치 |
| **구성 비율** | 브랜드(2) + 카테고리(5~7) + 도달 확장(5~7) + 니치(3~5) |
| **Reels 특이사항** | Reels는 해시태그보다 **트렌드 오디오**와 **첫 2초 후킹**이 도달에 더 중요 |
| **스토리** | 해시태그 스티커 1~2개 활용. 위치 태그와 함께 사용하면 도달 증가 |
| **알고리즘 팁** | 매번 동일한 해시태그 세트를 복붙하면 스팸으로 인식될 수 있음. 70% 고정 + 30% 교체 권장 |

**배치 예시 (Reels - 기술 소개):**
```
손만 움직여서 3D 모델을 조작한다?
DURU BIRD의 핸드트래킹 기술을 확인하세요!

프로필 링크에서 더 많은 정보를 만나보세요.

.
.
.

#파란두루미 #DURUBIRD #핸드트래킹 #XR솔루션 #확장현실
#디지털트윈 #혼합현실 #3D인터랙티브 #몰입형기술
#mixedreality #handtracking #immersivetech #XR
#테크 #스타트업 #혁신 #미래기술
```

### 3-3. TikTok

| 항목 | 전략 |
|---|---|
| **최적 개수** | 3~5개 (적을수록 좋음) |
| **배치 위치** | 캡션에 직접 포함 (자연스럽게 문장에 녹이거나 캡션 끝에 배치) |
| **핵심 원칙** | TikTok은 해시태그가 아니라 **콘텐츠 자체**로 추천됨. 해시태그는 카테고리 분류 용도 |
| **트렌드 해시태그** | TikTok 검색 탭에서 현재 인기 해시태그를 확인하고 관련 있으면 1개 포함 |
| **FYP 해시태그** | `#fyp` `#foryou` 등은 효과 미미. 사용하지 않아도 됨 |
| **알고리즘 팁** | 영상 완시율(끝까지 보는 비율)과 공유 수가 도달의 핵심. 해시태그 의존 금지 |

**배치 예시 (비하인드 영상):**
```
XR 스타트업의 월요일은 이런 느낌 #파란두루미 #XR #스타트업일상 #개발비하인드
```

### 3-4. LinkedIn

| 항목 | 전략 |
|---|---|
| **최적 개수** | 3~5개 (최대 5개 엄수) |
| **배치 위치** | 게시물 본문 마지막 줄에 배치 |
| **핵심 원칙** | LinkedIn은 **프로페셔널 키워드** 중심. 지나치게 캐주얼한 해시태그 사용 금지 |
| **영문 비율** | 글로벌 도달을 위해 영문 해시태그 50% 이상 권장 |
| **팔로우 가능 해시태그** | LinkedIn에서는 해시태그를 팔로우하는 기능이 있어, 업계 표준 해시태그 사용이 중요 |
| **알고리즘 팁** | LinkedIn 알고리즘은 **게시 후 1시간 내 반응**이 핵심. 해시태그보다 동료 태그와 공유가 효과적 |

**배치 예시 (성과 공유):**
```
CES 2026에서 DURU BIRD 핸드트래킹 기술을 선보였습니다.
현장에서 만난 50여 개 기업의 피드백을 바탕으로
더 나은 XR 솔루션을 만들어 나가겠습니다.

#Parandurume #CES2026 #XR #DigitalTwin #Innovation
```

### 3-5. 채널별 요약 비교표

| 채널 | 최적 개수 | 배치 위치 | 한/영 비율 | 핵심 포인트 |
|---|---|---|---|---|
| YouTube | 5~10개 | 설명란 | 한글 70% | 제목 키워드가 더 중요 |
| Instagram | 15~20개 | 캡션 하단 / 첫 댓글 | 한글 60% | 세트 교체하여 스팸 방지 |
| TikTok | 3~5개 | 캡션 내 | 한글 80% | 콘텐츠 품질이 핵심 |
| LinkedIn | 3~5개 | 본문 마지막 줄 | 영문 50%+ | 업계 표준 키워드 사용 |
| 네이버 블로그 | 10~15개 | 태그 기능 | 한글 100% | SEO 키워드 연동 필수 |

## 5. 네이버 블로그 SEO 키워드

네이버 블로그는 해시태그보다 **본문 내 키워드 배치**가 검색 노출(SEO)에 핵심적입니다.

### 5-1. 주력 키워드 (메인 키워드)

검색량이 높고 파란두루미의 핵심 사업과 직결되는 키워드입니다. 모든 블로그 포스트의 **제목에 1개 이상** 포함하세요.

| 키워드 | 월 검색량 수준 | 경쟁도 | 비고 |
|---|---|---|---|
| XR 솔루션 | 중 | 중 | 가장 핵심적인 키워드 |
| 디지털트윈 | 높음 | 높음 | 검색량 많지만 경쟁 치열 |
| 확장현실 | 중 | 중 | XR의 한글 표현 |

### 5-3. 키워드 배치 전략

블로그 포스트 내에서 키워드를 효과적으로 배치하는 방법입니다.

#### 제목 (가장 중요)

- **주력 키워드 1개를 반드시 포함**
- 제목 앞쪽에 키워드 배치 (네이버는 제목 앞부분에 더 큰 가중치)
- 50자 이내 권장

```
좋은 예: "디지털트윈 대시보드로 시설관리 효율 40% 높인 비결"
나쁜 예: "파란두루미가 알려주는 꿀팁! 효율적인 관리 방법"
```

#### 본문 구조

```
[서론] - 주력 키워드 1회 자연스럽게 언급 (첫 200자 이내)
[본론 1] - 롱테일 키워드 포함한 소제목 (H2/H3 태그 활용)
[본론 2] - 관련 키워드 자연스럽게 2~3회 분포
[결론] - 주력 키워드 1회 재언급 + CTA
```

#### 이미지 대체 텍스트 (ALT)

- 모든 이미지에 키워드 포함 대체 텍스트 기입
- 예: `파란두루미 DURU TWIN 디지털트윈 대시보드 화면`

#### 네이버 태그 기능

- 포스트 작성 시 하단 '태그' 필드에 **10~15개 키워드** 입력
- 주력 키워드 + 롱테일 키워드 + 브랜드명 조합
- 예: `파란두루미, DURU TWIN, 디지털트윈, 디지털트윈 대시보드, 시설관리, 스마트빌딩, XR 솔루션, 디지털전환, DX, 건물관리 솔루션`

### 5-4. 네이버 블로그 SEO 체크리스트

- [ ] 제목에 주력 키워드 1개 포함 (앞쪽 배치)

## 7. 해시태그 Do/Don't

### DO (올바른 사용법)

| 번호 | 가이드 | 설명 |
|---|---|---|
| 1 | **브랜드 해시태그를 항상 포함하세요** | `#파란두루미` 또는 `#Parandurume`은 모든 게시물의 필수 요소 |
| 2 | **채널별 최적 개수를 지키세요** | TikTok 3~5개, LinkedIn 3~5개, Instagram 15~20개, YouTube 5~10개 |
| 3 | **카테고리에 맞는 해시태그를 선택하세요** | 기술 소개 게시물에 채용 해시태그를 넣지 마세요 |
| 4 | **한글/영문을 적절히 혼합하세요** | 국내 도달은 한글, 글로벌 도달은 영문으로 병행 |
| 5 | **해시태그 세트를 주기적으로 교체하세요** | 동일 세트 반복 사용은 알고리즘에 불리. 70% 고정 + 30% 교체 |
| 6 | **트렌드 해시태그를 모니터링하세요** | 각 채널의 탐색/검색 탭에서 관련 인기 해시태그를 주기적으로 확인 |
| 7 | **띄어쓰기 없이 붙여 쓰세요** | `#디지털트윈` (O), `#디지털 트윈` (X - 해시태그가 끊김) |
| 8 | **해시태그 성과를 분석하세요** | Instagram 인사이트에서 해시태그별 도달 수를 월 1회 확인 |
| 9 | **니치(틈새) 해시태그도 활용하세요** | 게시물 수가 1만~10만인 중형 해시태그가 상위 노출에 유리 |
| 10 | **게시물 내용과 해시태그를 일치시키세요** | 해시태그는 콘텐츠 분류 도구. 관련 없는 태그는 역효과 |

### DON'T (피해야 할 패턴)

| 번호 | 가이드 | 설명 |
|---|---|---|
| 1 | **해시태그를 과도하게 넣지 마세요** | 채널별 권장 수를 크게 초과하면 스팸으로 분류될 위험 |
| 2 | **매번 같은 해시태그 세트를 복붙하지 마세요** | Instagram/TikTok 알고리즘이 스팸으로 판단하여 도달 제한 |
| 3 | **콘텐츠와 무관한 인기 해시태그를 사용하지 마세요** | `#먹스타그램` `#일상` 등 XR과 무관한 태그는 도달에 도움이 안 됨 |
| 4 | **금지(Banned) 해시태그를 사용하지 마세요** | Instagram에서 특정 해시태그가 금지 처리된 경우 게시물 전체 도달이 감소 |
| 5 | **`#fyp` `#foryou` `#viral` 등에 의존하지 마세요** | TikTok에서 이런 태그의 실제 효과는 입증되지 않음 |
| 6 | **해시태그만으로 도달을 기대하지 마세요** | 콘텐츠 품질, 게시 시간, 초반 반응이 알고리즘에 훨씬 중요 |
| 7 | **경쟁사 브랜드명을 해시태그로 쓰지 마세요** | `#○○사` 등 경쟁사 태그 사용은 비윤리적이며 법적 문제 소지 |
| 8 | **너무 긴 해시태그를 만들지 마세요** | `#파란두루미디지털트윈솔루션도입사례` 같은 태그는 검색 불가 |
| 9 | **해시태그를 본문 중간에 섞지 마세요** | 가독성을 해침. 본문과 해시태그는 분리 배치 |
| 10 | **특수문자를 해시태그에 넣지 마세요** | `#XR!솔루션` `#디지털-트윈` 등 특수문자 포함 시 태그가 끊김 |
